# Webshop: Runtime Records

In deze module bevat opdrachten behoren bij de webshop case [Runtime Records](https://e-learning.educom.nu/cases/runtime-records/intro)

## Module

Je gaat werken aan een applicatie op basis van OOP en MVC principes en met een database.

In deze module zie je een set aan [issues](../../issues) en dit is een overzicht van de opdrachten die je uit moet voeren. In deze module ga je de issues voornamelijk zelf aanvullen.
Herlees even de [werkinstructie](https://e-learning.educom.nu/algemeen/ProjectStructure/repository) voor het werken met GitHub issues binnen je projecten.
